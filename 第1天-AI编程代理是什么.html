AI编程代理到底是什么？

概念认知 · 零基础友好

💬 大白话解释

AI编程代理 = 会自己动手写代码、创建文件、执行命令、修bug的AI。

打个比方：

  • ChatGPT网页版 = 给你菜谱的服务员：你说"宫保鸡丁怎么做？"它把菜谱写给你，你还得自己去买菜、切菜、炒菜
  • Codex/Claude Code/OpenClaw = 全自动厨师机器人：你说"我要吃宫保鸡丁"，它自己去冰箱拿食材、切好、炒好、装盘——你只需尝一口说"OK"

[官方] 此比喻基于三个工具官方仓库 README 中对"autonomous coding agent"的定义。
[验证:2026-06-16] 已验证三个工具均支持自主创建文件+执行命令+修改代码的核心能力。

🔍 核心区别对比表

对比维度ChatGPT网页版AI编程代理
给代码✅ 给文字✅ 直接创建文件+写入代码
改文件❌ 不能动你的电脑✅ 可以直接读写修改文件
执行命令❌ 不能✅ 安装依赖、运行测试、启动服务
看结果自修复❌ 不知道代码对不对✅ 运行后自己读报错、修bug
你的角色搬运工（复制粘贴）审批官（审核+批准）

Windows安装Node.js

环境配置 · ⚠️ 三个工具共同的前置依赖

💬 大白话解释

Node.js 是让 JavaScript 脱离浏览器、直接在电脑上运行的环境。Codex CLI、Claude Code、OpenClaw 都用 JavaScript 写的——没有 Node.js 就等于没有操作系统，后面的全白搭。

🛠 安装步骤

1

先检查有没有装过：按 Win+R → 输入 powershell → 回车，键入：

PowerShell 
PS C:\Users\你的用户名> node --version
v22.22.2
PS C:\Users\你的用户名> npm --version
10.9.7

✅ 显示了版本号 → 已安装，跳到知识点3。

❌ 提示"不是内部或外部命令" → 继续步骤2。

2

去官网下载安装

浏览器打开 https://nodejs.org

  • 点左边绿色 LTS 按钮（当前推荐 Node 22）
  • 下载 .msi 安装包
  • 双击运行，一路点 "Next"，全部默认选项
  • 安装完成后必须关掉所有 PowerShell/CMD 窗口，重新打开
  • 重复步骤1验证

3

npmmirror国内镜像配置

不用翻墙 · 下载npm包速度翻10倍

💬 大白话解释

npm 默认从国外服务器下载包，国内慢得像蜗牛。换成淘宝镜像（npmmirror.com）后，所有包走国内 CDN，下载速度从几十KB/s变成几十MB/s。

🛠 配置步骤

1

一行命令搞定：

PowerShell 
PS> npm config set registry https://registry.npmmirror.com
2

验证是否生效：

PowerShell 
PS> npm config get registry
https://registry.npmmirror.com

✅ 输出 npmmirror.com 即配置成功。

⚠️ 注意

如果之后想换回官方源：npm config set registry https://registry.npmjs.org

[官方] 淘宝npm镜像官方站: npmmirror.com — 由阿里云维护的npm完整镜像。
